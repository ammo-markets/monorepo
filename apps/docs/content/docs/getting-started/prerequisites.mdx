Before you can mint or redeem ammunition tokens, you'll need a few things set up.

## Crypto Wallet

You need a browser-based crypto wallet. Recommended wallets:

- **MetaMask** — the most widely used browser wallet
- **Rabby** — developer-friendly with multi-chain support
- **Coinbase Wallet** — good for beginners
- **WalletConnect** — connects mobile wallets to desktop

AmmoMarkets uses [RainbowKit](https://www.rainbowkit.com/) for wallet connection, so most popular EVM wallets are supported.

## Gas Fees

You need a small amount of the network's native token to pay for transaction fees (gas). Typical transactions cost less than $0.10.

## Network Setup

AmmoMarkets runs on an EVM-compatible blockchain. Most wallets will prompt you to add the correct network automatically when you connect. Simply approve the network switch when prompted.
