## One Token Per Caliber + Load Type

Ammo Exchange creates a separate ERC-20 token for each combination of caliber and load type. This means practice ammunition and self-defense ammunition for the same caliber are different tokens with different prices.

Why? Practice and self-defense rounds have fundamentally different costs, demand patterns, and use cases. Separating them ensures accurate pricing and lets you choose exactly what you want exposure to.

## Currently Available Tokens

| Token | Caliber | Load Type | Description |
|-------|---------|-----------|-------------|
| **ax9P** | 9mm Luger | Practice | 115gr FMJ brass-case — the standard range round |
| **ax9SD** | 9mm Luger | Self-Defense | 124gr JHP — jacketed hollow point for carry |
| **ax556P** | 5.56 NATO | Practice | 55gr FMJ — standard rifle range ammo |
| **ax556SD** | 5.56×45mm | Self-Defense | 62gr soft-point — defensive rifle round |

Each token equals **1 round** of its respective ammunition type.

## Token Properties

- **Standard ERC-20** — compatible with any EVM wallet, DEX, or DeFi protocol
- **18 decimals** — divisible for precise trading (though you can only redeem whole rounds)
- **Mintable/Burnable** — minted when users deposit stablecoins, burned when users redeem for physical ammo
- **Transferable** — you can send tokens to any wallet address
