Add a global focus-visible rule in `@layer base` for raw interactive elements that don't use the shadcn Button component:

```css
/* Focus-visible for raw interactive elements not using shadcn Button */
a:focus-visible,
button:focus-visible,
[role="button"]:focus-visible,
[role="tab"]:focus-visible,
input:focus-visible,
select:focus-visible,
textarea:focus-visible {
  outline: 2px solid var(--ring);
  outline-offset: 2px;
  border-radius: var(--radius-sm);
}
```

  <action>
For EVERY file listed, replace ALL onMouseEnter/onMouseLeave handler pairs with equivalent Tailwind hover: classes. The pattern is mechanical:

**Common hover patterns and their Tailwind replacements:**

1. **Background color change on hover** (most common):
   ```
   BEFORE:
   style={{ backgroundColor: "var(--brass)" }}
   onMouseEnter={(e) => { e.currentTarget.style.backgroundColor = "var(--brass-hover)"; }}
   onMouseLeave={(e) => { e.currentTarget.style.backgroundColor = "var(--brass)"; }}

   AFTER:
   className="... bg-brass hover:bg-brass-hover"
   (remove style.backgroundColor, remove onMouseEnter, remove onMouseLeave)
   ```

2. **Background + border change on hover**:
   ```
   BEFORE:
   style={{ backgroundColor: "transparent", border: "1px solid var(--border-hover)" }}
   onMouseEnter={(e) => {
     e.currentTarget.style.backgroundColor = "var(--bg-tertiary)";
     e.currentTarget.style.borderColor = "var(--brass-border)";
   }}
   onMouseLeave={(e) => {
     e.currentTarget.style.backgroundColor = "transparent";
     e.currentTarget.style.borderColor = "var(--border-hover)";
   }}

   AFTER:
   className="... bg-transparent hover:bg-ax-tertiary border border-border-hover hover:border-brass-border"
   (remove style.backgroundColor, style.border, onMouseEnter, onMouseLeave)
   ```

3. **Opacity/color change on hover**:
   ```
   BEFORE:
   style={{ color: "var(--text-muted)" }}
   onMouseEnter={(e) => { e.currentTarget.style.color = "var(--text-primary)"; }}
   onMouseLeave={(e) => { e.currentTarget.style.color = "var(--text-muted)"; }}

   AFTER:
   className="... text-text-muted hover:text-text-primary"
   ```

4. **Brass muted background hover** (common for subtle buttons):
   ```
   BEFORE:
   style={{ backgroundColor: "rgba(198, 164, 78, 0.1)" }} in onMouseEnter

   AFTER:
   className="... hover:bg-brass-muted"
   (brass-muted is rgba(198, 164, 78, 0.15) -- close enough, consistent with design system)
   ```

5. **Amber hover** (wrong network button):
   ```
   BEFORE:
   onMouseEnter: backgroundColor = "rgba(245, 158, 11, 0.1)"

   AFTER:
   className="... hover:bg-ammo-amber/10"
   ```

- Add `transition-colors duration-150` class if not already present (most elements already have it)
