  <action>
Refactor the "State D" (connected + signed in) section of WalletButton to use a DropdownMenu instead of a plain button that instantly disconnects.

**Imports to add:**

- `{ Copy, ExternalLink, LogOut }` from `lucide-react`
- `{ DropdownMenu, DropdownMenuTrigger, DropdownMenuContent, DropdownMenuItem, DropdownMenuSeparator }` from `@/components/ui/dropdown-menu`
- `{ AlertDialog, AlertDialogAction, AlertDialogCancel, AlertDialogContent, AlertDialogDescription, AlertDialogFooter, AlertDialogHeader, AlertDialogTitle }` from `@/components/ui/alert-dialog`
- `{ useState }` from `react`
- `{ AVALANCHE_FUJI }` from `@ammo-exchange/shared` (for blockExplorers.default)

**State D replacement:**

1. Add `const [showDisconnectDialog, setShowDisconnectDialog] = useState(false)` at component top
2. Wrap the existing wallet display button in `<DropdownMenu>` / `<DropdownMenuTrigger asChild>`
3. The trigger button keeps the same styling as current State D button but removes the onClick handler (dropdown handles it now). Keep the identicon, truncated address, and USDC balance display exactly as-is. Keep `aria-label="Wallet menu"` (changed from "Disconnect wallet").
4. Add `<DropdownMenuContent align="end">` with three items:
   - **Copy Address**: `<DropdownMenuItem>` with `<Copy size={16} />` icon and "Copy Address" text. onClick: `navigator.clipboard.writeText(address!)` then optionally show nothing (keep it simple, no toast needed).
   - **View on Explorer**: `<DropdownMenuItem>` with `<ExternalLink size={16} />` icon and "View on Explorer" text. onClick: `window.open(\`${AVALANCHE_FUJI.blockExplorers.default}/address/${address}\`, '\_blank')`.
   - `<DropdownMenuSeparator />`
   - **Disconnect**: `<DropdownMenuItem variant="destructive">` with `<LogOut size={16} />` icon and "Disconnect" text. onClick: `setShowDisconnectDialog(true)`.

5. After the `</DropdownMenu>`, add an `<AlertDialog open={showDisconnectDialog} onOpenChange={setShowDisconnectDialog}>` with:
   - `<AlertDialogContent>` containing:
     - `<AlertDialogHeader>` with `<AlertDialogTitle>Disconnect Wallet?</AlertDialogTitle>` and `<AlertDialogDescription>This will sign you out and disconnect your wallet. You'll need to reconnect and sign in again to continue.</AlertDialogDescription>`
     - `<AlertDialogFooter>` with `<AlertDialogCancel>Cancel</AlertDialogCancel>` and `<AlertDialogAction variant="destructive" onClick={() => { signOut(); disconnect(); }}>Disconnect</AlertDialogAction>`

**Important:** States A (disconnected), B (wrong network), and C (not signed in) remain exactly as they are -- only State D changes.
