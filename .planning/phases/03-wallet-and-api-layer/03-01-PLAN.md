    **1. Create `apps/web/hooks/use-wallet.ts`:**

    A `"use client"` hook composing wagmi hooks into a single application-specific interface. Import `useAccount`, `useConnect`, `useDisconnect`, `useSwitchChain` from `wagmi` and `injected` from `wagmi/connectors` and `avalancheFuji` from `wagmi/chains`.

    Return an object with:
    - `address` (from useAccount)
    - `isConnected` (from useAccount)
    - `isReconnecting` (from useAccount -- used to prevent hydration mismatch)
    - `isWrongNetwork`: true when `isConnected && chainId !== avalancheFuji.id`
    - `chainId` (from useAccount)
    - `connect`: calls `connect({ connector: injected() })`
    - `disconnect`: calls `disconnect()`
    - `switchToFuji`: calls `switchChain({ chainId: avalancheFuji.id })`
    - `isConnecting` (isPending from useConnect)
    - `isSwitching` (isPending from useSwitchChain)

    Follow the exact pattern from research Pattern 1 / Code Examples section. Export the hook as a named export.

    **2. Create `apps/web/hooks/use-token-balances.ts`:**

    A `"use client"` hook that reads USDC balance + 4 AmmoToken balances in a single multicall using `useReadContracts` from wagmi. Import `erc20Abi` from `viem` for the USDC balance read. Import `AmmoTokenAbi` from `@ammo-exchange/contracts/abis`. Import `CONTRACT_ADDRESSES` from `@ammo-exchange/shared`. Import `type Caliber` from `@ammo-exchange/shared`.

    Define `const CALIBERS: Caliber[] = ["9MM", "556", "22LR", "308"]` and `const fuji = CONTRACT_ADDRESSES.fuji`.

    Build a contracts array with 5 entries:
    - Entry 0: `{ address: fuji.usdc, abi: erc20Abi, functionName: "balanceOf", args: [address!] }`
    - Entries 1-4: map over CALIBERS to create `{ address: fuji.calibers[caliber].token, abi: AmmoTokenAbi, functionName: "balanceOf" as const, args: [address!] }`

    Pass `query: { enabled: isConnected && !!address }` to prevent calls when disconnected.

    Return:
    - `usdc`: `data?.[0]?.result as bigint | undefined`
    - `tokens`: `Record<Caliber, bigint | undefined>` built from `Object.fromEntries(CALIBERS.map(...))`
    - `isLoading`
    - `refetch`

    Follow the exact pattern from research Pattern 2. Use `useAccount` from wagmi to get address and isConnected.

    **3. Add utilities to `apps/web/lib/utils.ts`:**

    Append to the existing file (keep the existing `cn` function). Add:
    - `truncateAddress(address: string): string` -- returns `${address.slice(0, 6)}...${address.slice(-4)}`
    - `snowtraceUrl(txHash: string): string` -- returns `https://testnet.snowtrace.io/tx/${txHash}`
    - `snowtraceAddressUrl(address: string): string` -- returns `https://testnet.snowtrace.io/address/${address}`

    Follow the exact patterns from research Code Examples "Address Truncation Utility" section.
