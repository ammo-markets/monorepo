**1. `apps/web/lib/errors.ts`** -- Contract error to human message mapping.

Create a `CONTRACT_ERROR_MESSAGES` record mapping all 12+ CaliberMarket custom error names to user-friendly strings:
- `InvalidAmount` -> "Amount must be greater than zero."
- `MinMintNotMet` -> "Amount is below the minimum mint requirement for this caliber."
- `MarketPaused` -> "This market is currently paused. Please try again later."
- `DeadlineExpired` -> "The transaction deadline has passed. Please submit a new order."
- `Slippage` -> "Price moved beyond your slippage tolerance. Try again with a higher slippage."
- `InvalidPrice` -> "Oracle price is unavailable. Please try again shortly."
- `TreasuryNotSet` -> "Protocol configuration error. Please contact support."
- `NotKeeper` -> "Unauthorized: only protocol keepers can execute this."
- `NotOwner` -> "Unauthorized: only protocol owner can execute this."
- `Reentrancy` -> "Transaction conflict detected. Please try again."
- `ZeroAddress` -> "Invalid address configuration."
- `InvalidBps` -> "Fee configuration error."
- `InsufficientBalance` -> "Insufficient token balance for this operation."
- `InsufficientAllowance` -> "Token allowance not set. Please approve first."

Export `parseContractError(error: Error | null): string` that:
1. Returns "" if error is null
2. Checks `(error as BaseError).shortMessage` for "User rejected" or "User denied" -> return "Transaction cancelled. You rejected the request in your wallet."
3. Inspects `error.cause?.data?.errorName` for custom error names -> return mapped message
4. Falls back to `baseError.shortMessage` if available
5. Default: "An unexpected error occurred. Please try again."

Import `type { BaseError }` from "wagmi". Add `"use client"` directive.

**2. `apps/web/lib/tx-utils.ts`** -- Transaction utility helpers.

Export these functions (NO "use client" needed -- pure utility):
- `getDeadline(hoursFromNow = 24): bigint` -- `BigInt(Math.floor(Date.now() / 1000) + hoursFromNow * 3600)`
- `DEFAULT_SLIPPAGE_BPS = BigInt(500)` (5%)
- `parseUsdc(amount: string): bigint` -- `parseUnits(amount, 6)` from viem
- `formatUsdc(amount: bigint): string` -- `formatUnits(amount, 6)` from viem
- `parseTokenAmount(rounds: string): bigint` -- `parseUnits(rounds, 18)`
- `formatTokenAmount(amount: bigint): string` -- `formatUnits(amount, 18)`

Import `parseUnits`, `formatUnits` from "viem".

**3. `apps/web/hooks/use-allowance.ts`** -- ERC20 allowance read hook.

Add `"use client"` directive.

Export `useAllowance(tokenAddress, owner, spender)` where:
- `tokenAddress: \`0x${string}\``
- `owner: \`0x${string}\` | undefined`
- `spender: \`0x${string}\``

Uses `useReadContract` from "wagmi" with `erc20Abi` from "viem":
- `address: tokenAddress`, `abi: erc20Abi`, `functionName: "allowance"`
- `args: owner ? [owner, spender] : undefined`
- `query: { enabled: !!owner }`

Returns `{ allowance: bigint | undefined, refetch, hasEnoughAllowance: (required: bigint) => boolean }`.
The `hasEnoughAllowance` checks `allowance !== undefined && allowance >= required`.

Cast the `data` from useReadContract `as bigint | undefined` for the allowance return.

**1. `apps/web/hooks/use-mint-transaction.ts`** -- Two-step approve + startMint hook.

Add `"use client"` directive.

Import: `useWriteContract`, `useWaitForTransactionReceipt` from "wagmi"; `erc20Abi`, `parseUnits` from "viem"; `CaliberMarketAbi` from "@ammo-exchange/contracts/abis"; `CONTRACT_ADDRESSES` from "@ammo-exchange/shared"; `type { Caliber }` from "@ammo-exchange/shared".

Export `useMintTransaction(caliber: Caliber)`:
- Derive `marketAddress = CONTRACT_ADDRESSES.fuji.calibers[caliber].market`
- Create TWO separate `useWriteContract` instances (critical -- do NOT share one instance for approve and mint):
  - Approve instance: `{ data: approveHash, error: approveError, isPending: isApprovePending, writeContract: writeApprove, reset: resetApprove }`
  - Mint instance: `{ data: mintHash, error: mintError, isPending: isMintPending, writeContract: writeMint, reset: resetMint }`
- Create TWO `useWaitForTransactionReceipt` instances:
  - `{ isLoading: isApproveConfirming, isSuccess: isApproveConfirmed } = useWaitForTransactionReceipt({ hash: approveHash })`
  - `{ isLoading: isMintConfirming, isSuccess: isMintConfirmed } = useWaitForTransactionReceipt({ hash: mintHash })`

Methods:
- `approve(usdcAmount: string)` -- calls `writeApprove({ address: CONTRACT_ADDRESSES.fuji.usdc, abi: erc20Abi, functionName: "approve", args: [marketAddress, parseUnits(usdcAmount, 6)] })`. NOTE: approve targets the USDC contract address, NOT the market address. The spender (args[1]) is the market address.
- `startMint(usdcAmount: string, slippageBps: bigint, deadline: bigint)` -- calls `writeMint({ address: marketAddress, abi: CaliberMarketAbi, functionName: "startMint", args: [parseUnits(usdcAmount, 6), slippageBps, deadline] })`
- `reset()` -- calls both `resetApprove()` and `resetMint()`

Return all state + methods: `{ approve, approveHash, approveError, isApprovePending, isApproveConfirming, isApproveConfirmed, startMint, mintHash, mintError, isMintPending, isMintConfirming, isMintConfirmed, reset }`.
