    **1. Create `app/api/admin/stats/route.ts`:**
    - Import publicClient from @/lib/viem
    - Import AmmoManagerAbi, AmmoTokenAbi from @ammo-exchange/contracts/abis
    - Import CONTRACT_ADDRESSES, CALIBER_SPECS from @ammo-exchange/shared
    - Import erc20Abi, formatUnits from viem
    - Import prisma from @ammo-exchange/db
    - Define CALIBERS array: ["9MM", "556", "22LR", "308"] as const
    - GET handler:
      1. Read treasury address: `publicClient.readContract({ address: CONTRACT_ADDRESSES.fuji.manager, abi: AmmoManagerAbi, functionName: "treasury" })`
      2. Read USDC balance of treasury: `publicClient.readContract({ address: CONTRACT_ADDRESSES.fuji.usdc, abi: erc20Abi, functionName: "balanceOf", args: [treasury] })`
      3. Read totalSupply for each caliber token (Promise.all with readContract, catch → BigInt(0)):
         ```
         const supplies = await Promise.all(
           CALIBERS.map(caliber =>
             publicClient.readContract({
               address: CONTRACT_ADDRESSES.fuji.calibers[caliber].token,
               abi: AmmoTokenAbi,
               functionName: "totalSupply",
             }).catch(() => BigInt(0))
           )
         );
         ```
      4. Query total redeemed from DB: `prisma.order.count({ where: { type: "REDEEM", status: "COMPLETED" } })`
      5. Query total minted from DB: `prisma.order.count({ where: { type: "MINT", status: "COMPLETED" } })`
      6. Query pending orders count: `prisma.order.count({ where: { status: "PENDING" } })`
    - Return JSON response:
      ```
      {
        treasuryUsdc: formatUnits(usdcBalance, 6),
        totalRedeemed,
        totalMinted,
        pendingOrders,
        calibers: CALIBERS.map((caliber, i) => ({
          caliber,
          name: CALIBER_SPECS[caliber].name,
          totalSupply: Math.floor(Number(formatUnits(supplies[i]!, 18))),
        })),
      }
      ```
    - Wrap in try/catch, return 502 on upstream error
